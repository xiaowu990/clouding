
# 基本语法
[ condition ]（注意 condition 前后要有空格） #非空返回 true，可使用$?验证（0 为 true，>1 为 false）

# 应用实例
[ atguigu ]              返回 true 
[] 返回 false 
[condition] && echo OK || echo notok 条件满足，执行后面的语句
# 常用判断条件
1. 两个整数的比较 
* = 字符串比较 
* -lt 小于 
* -le 小于等于
* -eq 等于 
* -gt 大于 
* -ge 大于等于
* -ne 不等于 

2. 按照文件权限进行判断 
* -r 有读的权限 [ -r 文件 ] 
* -w 有写的权限 
* -x 有执行的权限

3. 按照文件类型进行判断 
* -f 文件存在并且是一个常规的文件 
* -e 文件存在 
* -d 文件存在并是一个目录
