# 1.统计行数
使用wc命令

`wc`命令的基本语法如下：

```
wc [选项] [文件]
```

常用选项：

- `-l`：仅统计行数。
- `-w`：仅统计字数（由空白字符分隔的词数）。
- `-c`：仅统计字节数。
- `-m`：仅统计字符数。
- `-L`：显示最长行的长度。

## 解法1-WC
1. 将文件作为参数传递给wc
wc -l nowcoder.txt
2. 使用重定向将文件通过标准输入传递给wc
wc -l > nowcoder.txt
3. 使用`cat`命令将文件`file.txt`的内容输出到标准输出
cat nowcoder.txt | wc -l
