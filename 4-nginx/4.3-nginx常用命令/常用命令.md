
# Nginx常用命令
使用这些命令时，要进入sbin中

进入 nginx 目录中: cd /usr/local/nginx/sbin

## 查看 nginx 版本号

./nginx -v

## 启动 nginx

./nginx

查看是否启动成功：ps -ef|grep nginx

![[Pasted image 20230730231633.png]]
## 停止 nginx

./nginx -s stop
![[Pasted image 20230730231556.png]]

## 重新加载 nginx
重加载：不需要重启，将配置更改后更新，==也叫做热部署==

./nginx -s reload
